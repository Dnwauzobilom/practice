Introduction to html

Talking about all that I've learnt in HTML

HTML stands for Hypertext Markup Language which uses elements to tell the browser how to display its content.

Html Building Blocks

According to the tutor, there are 3 major basic building blocks which include the following: *Tags *Attributes *Elements

HTML FORMATTING

There are so many html formatting elements which are designed to specially identify some texts which the designer wants to point out specially.

To mention few of these formatting elements, I'll also attach the function of those elements mentioned. These elements include the following: Bold text, Important text, Italic text, Emphasized text, Marked text, Smaller text, Deleted text, Inserted text, Subscript Text, Superscript Text.

It is worthy of note that for every opening tag, there must be a corresponding closing tag and a closing tag is identified by a forward-slash (/) in front of the opening tag which converts the opening tag into a closing tag.

HTML HEADINGS

In HTML, there are six(6) different heading tags ranging from h1 to h6

. h1 is the largest heading tag, and h6 is the smallest heading tag. h1 being the largest heading tag implies that h1 is used for the most important heading while h6 is used for the least important heading.
